# Accessibility-Driven Web Content Production 

# Overview
This CISC4900 Project repository documents my internship project whose focus invloves web accesibility, content systems optimization, and SEO implementation for Brooklyn College website. My collaboration with the college's Marketing & Communications department includes audit, remediation, and optimization of web pages to improve accessibility, usability, and search visibility. 

The goal of my collaboration is to ensure the website is alligned with the principles of accessibility (perceivable, operable, understandable, robust), by improving the quality, usability, and accessibility of content for all kinds of users. Through ensuring links are correct, pages are consistently built and organized, pages are findable by search engines.
